Test Example

This is a test website to show what CHTTPD is capable of.

CHTTPD is written in C from scratch using threads (I don't like non-blocking I/O) and raw sockets.

Unfortunately, this can lead to Slowloris attacks due to limited threads.

It is not meant for production (it is insecure, full of memory leaks!), and it is only a proof of concept for myself.

For instance, if you refresh quickly enough, you'll get a double free().
